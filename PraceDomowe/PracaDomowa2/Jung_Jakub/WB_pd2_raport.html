4. Porwnanie z wyjanieniami BreakDown i SHAP

Dla przypomnienia - LIME:

pp_lime_9 <- predict_surrogate(
  explainer=explainer, 
  new_observation=diabetes[9,-9], 
  n_features=3, 
  n_permutations=1000,
  type='lime')

plot(pp_lime_9)

BreakDown:

pp_bd_9 <- predict_parts(explainer, new_observation = diabetes[9,], type = "break_down")
plot(pp_bd_9)

SHAP:

pp_shap_9 <- predict_parts(explainer, new_observation = diabetes[9,], type = "shap", B = 10)
plot(pp_shap_9)

Najwaniejsze zmienne w rnych wyjanieniach pokrywaj si, rnic si co najwyej kolejnoci i wag.
